# Expense Tracker (Трекер расходов)

**Автор:** Тилов Хызыр

## Описание программы

Expense Tracker - это графическое приложение для отслеживания личных расходов. Программа позволяет добавлять расходы с указанием суммы, категории и даты, просматривать список всех расходов в удобной таблице, фильтровать расходы по категории или дате, подсчитывать сумму расходов за выбранный период, удалять ненужные записи и сохранять данные между запусками программы.

## Как запустить приложение

1. Убедитесь, что у вас установлен Python версии 3.6 или выше. Проверить версию можно командой: python --version

2. Скачайте или клонируйте репозиторий:
   git clone https://github.com/parklir/Expense-Tracker.git
   cd expense-tracker

3. Запустите программу командой:
   python main.py

Никакие дополнительные библиотеки не требуются. Программа использует только стандартную библиотеку Python.

## Как использовать программу

### Добавление расхода

В поле "Сумма (руб)" введите сумму, например 350.50. Выберите категорию из выпадающего списка: Еда, Транспорт, Развлечения, Коммунальные, Здоровье или Другое. В поле "Дата" введите дату в формате ДД-ММ-ГГГГ, например 09-05-2025. Нажмите кнопку "Добавить расход". Расход появится в таблице и автоматически сохранится в файл.

### Фильтрация расходов

Для фильтрации по категории выберите нужную категорию в блоке "Фильтрация" и нажмите кнопку "Применить". В таблице останутся только расходы выбранной категории.

Для фильтрации по дате введите дату в поле "Дата" в блоке фильтрации и нажмите "Применить". Останутся расходы только за указанную дату.

Чтобы сбросить фильтры, нажмите кнопку "Сбросить". Таблица снова покажет все расходы.

### Подсчёт суммы за период

В поле "От" введите начальную дату в формате ДД-ММ-ГГГГ, например 01-05-2025. В поле "До" введите конечную дату, например 31-05-2025. Нажмите кнопку "Рассчитать". Внизу блока отобразится общая сумма расходов за указанный период.

### Удаление расхода

Выделите нужную строку в таблице, кликнув по ней мышью. Нажмите кнопку "Удалить выбранный". Подтвердите удаление во всплывающем окне.

### Общая сумма

В правом нижнем углу экрана отображается общая сумма всех расходов с учётом применённых фильтров.

## Примеры использования

Пример 1: Добавление нескольких расходов

Добавьте следующие расходы:
Дата 09-05-2025, категория Еда, сумма 350.50
Дата 10-05-2025, категория Транспорт, сумма 100.00
Дата 11-05-2025, категория Развлечения, сумма 500.00

После добавления в правом нижнем углу отобразится: Общая сумма: 950.50 руб.

Пример 2: Фильтрация по категории

Выберите категорию "Еда" в блоке фильтрации и нажмите "Применить". В таблице останутся только расходы на еду. Общая сумма изменится на 350.50 руб.

Пример 3: Подсчёт суммы за месяц

В поле "От" введите 01-05-2025, в поле "До" введите 31-05-2025. Нажмите "Рассчитать". Результат: Сумма за период 01-05-2025 - 31-05-2025: 950.50 руб.

## Формат хранения данных

Расходы сохраняются в файл expenses.json в той же папке, где находится программа. Пример содержимого файла:

[
    {
        "amount": 350.5,
        "category": "Еда",
        "date": "09-05-2025"
    },
    {
        "amount": 100,
        "category": "Транспорт",
        "date": "10-05-2025"
    },
    {
        "amount": 500,
        "category": "Развлечения",
        "date": "11-05-2025"
    }
]
